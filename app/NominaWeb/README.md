# Sistema de Nómina (ASP.NET Core MVC)

Proyecto desarrollado para la asignatura Integración de Software.

## Tecnologías
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Razor Views
- Git / GitHub

## Funcionalidades
- CRUD de Empleados
- CRUD de Departamentos
- Registro de Salarios
- Auditoría de cambios de salario
- Validación de fechas (Hasta >= Desde)
- Prevención de solapamiento de salarios por empleado
- Validación de existencia de empleado al registrar salario
